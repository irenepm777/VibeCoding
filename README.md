# VibeCoding


## PROMPT 1 - Estructura base de la landing

Quiero crear una landing page divulgativa sobre el Mesozoico, accesible desde internet, utilizando únicamente HTML, CSS y JavaScript vanilla (sin frameworks).

La web debe tener una estructura clara y sencilla, pensada como base para futuras mejoras.

Requisitos:
- Una landing page informativa dividida en las tres grandes etapas del Mesozoico: Triásico, Jurásico y Cretácico.
- Cada sección debe incluir un título, una breve descripción de la época y algunos datos generales (clima, fauna dominante, cambios relevantes).
- Incluir una sección inicial de introducción que explique qué es el Mesozoico.
- Diseño limpio y legible, sin una estética muy elaborada todavía.
- La web debe ser responsive, adaptándose correctamente a distintos tamaños de pantalla.
- Código bien estructurado y comentado para facilitar modificaciones posteriores.

El objetivo es obtener una primera versión funcional y ordenada de la landing, que sirva como punto de partida para añadir interactividad y mejoras visuales más adelante.


## PROMPT 2 - Calculadora y responsive

Partiendo de una landing page ya creada sobre el Mesozoico, quiero mejorarla añadiendo interactividad y funcionalidad, manteniendo el uso exclusivo de HTML, CSS y JavaScript vanilla (sin frameworks).

Requisitos:
- Mantener la estructura general de la landing (introducción + Triásico, Jurásico y Cretácico).
- Añadir una sección interactiva con una calculadora donde el usuario pueda introducir su edad, altura y peso.
- La calculadora debe validar los datos introducidos y no aceptar valores imposibles o incoherentes.
- En función de los datos introducidos, la web debe mostrar una estimación del tiempo de supervivencia del usuario en cada una de las tres eras del Mesozoico.
- Mostrar los resultados de forma clara y visual, permitiendo comparar las tres eras.
- Mejorar el diseño responsive para que la calculadora y los resultados sean usables tanto en escritorio como en dispositivos móviles.
- Mantener un diseño limpio y legible, sin una experiencia visual inmersiva avanzada todavía.

El objetivo es obtener una segunda versión más interactiva de la landing, centrada en la funcionalidad y la experiencia de usuario, que sirva de base para futuras mejoras estéticas e inmersivas.
